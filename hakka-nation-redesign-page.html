An in-class assignment where, using Figma, I redesigned the Hakka Nation restaurant's website, improving aesthetics and improve overall usability.

The main focus of this project was UX improvement. You can view the Figma design file here. If the process is what interests you, you’ll find it bellow the embed.

1 - Sitemaps

To begin, I took inventory of the official site [view here] and mapped out how each page connects alongside important headings.

Additionally, I took note of pain points I found while browsing the site. Things like their gallery at the bottom of the page with no descriptions, etc.

Organizing my thoughts, I recreated the sitemap to alleviate as many pain points as I could, and only then began work redesigning four of the main pages.

2 - Spotlight

Focusing in on my favorite part of the redesign, I believe the menu succeeds in organizing the crazy amount of information thrown into the original. More than that, makes using it pleasant. What do I mean by that?

First, the image below is a screenshot of the official restaurant's Waterloo location menu.
Inspiring isn’t it?

Now, here's some screenshots of my redesign.

By making a list of pain points before starting work, I was able to address them wherever they reared their head. For starters, in the images above I’ve added textures and pictures to fill the void present in the original.

Additionally, I reorganized the text to clarify what meant what, and remove some redundancy. No longer will the menu list the toppings three times.

3 - Closing Notes

If I were to go back and redo this today, I’d want to improve on two points.

Primarily, I would allocate time to map out the user flow. Having a good setup means nothing if it’s not how people want to use it. Some Figma interactions paired with a round of user testing would’ve pushed this redesign further.

I would like to make use of a slightly wider colour palette, rather than the muted yellows and greys. Their food is vibrant (and really good).
The website should reflect that.

All things considered, I learned plenty from this project and seeing how my design knowledge stacks up was encouraging.
